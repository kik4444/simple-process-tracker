Simple Process Tracker v1.0.2


A lightweight, simple Qt-based app that tracks how long certain processes have been running for.

Source code - https://github.com/kik4444/simple-process-tracker


How to use


Adding processes to track

Click the "Add" button on the main window. A new entry will appear in the list. Click the "select" button on the left and a dialog will appear with all processes that are currently running on this computer. Scroll to the one you want to track or write its name to filter the results. Select the one you wish to track and it will appear in the list. Now, whenever the process is running, its run time will count upwards. If the process is not running its run time will not increase.


Add custom icon to process

Click on the icon to the left of the process name and select a picture somewhere on your computer.


Set custom running duration

Click on the run time counter on the right of the process name and an input dialog will appear. Write your custom duration in HH:MM:SS format. E.g. 1 hour, 32 minutes and 42 seconds is 01:32:42. Click "Ok" and the processes' duration will start counting up from that.


Pause / resume time tracking

Uncheck the "tracking" button and the process will no longer be tracked. Check it again to resume tracking it.


Hide / unhide processes

Click the "Hide" button to both disable and hide a process from the list. This is useful for archiving processes you no longer wish to see on the main list. If you wish to unhide and resume tracking it, click the "Show hidden" button on the main window, locate the process and click its "Unhide" button.


Removing processes

If you wish to permanently remove a process from the list, click the "Remove" button and confirm. This will also delete all recorded duration for this process.


Options

Click the "Options" button on the main window to open the options window. Here you can set how often Simple Process Tracker should check whether a process is still running or not. Default values for running and stopped processes is 5 and 10 seconds respectively. This means that when a process starts, at most 10 seconds will pass before Simple Process Tracker starts tracking it. On the other hand, when a process closes, at most 5 seconds will pass before it stops being tracked. Lower values = more accuracy + more CPU usage. Higher values = lower accuracy + lower CPU usage.


Manually check processes

Click the "Poll" button on the main window button to force check processes. This is useful if you've set process checking to be very infrequent in the options window.


Running in the background

This program can work in the background if you close the main window. It is visible in the system tray icon. To close the program completely, right click the system tray icon and click "exit" or the big "exit" button on the main window.
